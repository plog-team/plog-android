# PLOG Android

모바일 프로그래밍 팀 프로젝트 **PLOG**의 Android 저장소입니다.  
PLOG는 **Photo + Log**의 합성어로, 사진 기반 일기 작성과 개인화 기능을 제공하는 모바일 애플리케이션입니다.

---

## 프로젝트 소개

- **프로젝트명**: PLOG
- **한 줄 소개**: 사진을 기반으로 하루를 기록하고, 이를 바탕으로 개인화된 기능을 제공하는 일기 앱
- **팀명**: Team 2
- **개발 환경**
  - Android Studio
  - Java (JDK 17)
  - Spring Boot
  - MySQL
  - GitHub

---

## 주요 기능

- 사진 업로드 기반 일기 작성
- 날짜 / 위치 / 날씨 자동 입력
- AI 가이드 기반 작성 보조
- 비밀글 / 북마크 / 임시저장 / 수정
- 감정-장소 분석 리포트
- 사진 기반 사용자 성향 및 경로 분석
- 공공데이터 API 연동 추천 기능
- 교환일기
- 검색 기능
- 재방문 / 일기 작성 알림
- AI 챗봇

## 기본 브랜치 안내

이 저장소의 **기본 작업 기준 브랜치는 `main`이 아니라 `dev`** 입니다.

- `main` : 최종 발표 / 배포용 안정 브랜치
- `dev` : 개발 통합 브랜치
- 기능 개발 및 버그 수정은 반드시 `dev`에서 분기한 브랜치에서 작업합니다.
- 모든 Pull Request의 기본 대상 브랜치는 `main`이 아니라 **`dev`** 입니다.
